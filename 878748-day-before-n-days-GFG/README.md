# [Day before N days](https://www.geeksforgeeks.org/problems/days-before-n-days--150030/1?page=1&category=python&sortBy=submissions)
## Easy
Given two integers d and n.&nbsp;Where d is the day, out of 7 days of the week, d varies from 0 to 6 as shown below.0 - Sunday1 -&nbsp;Monday2 -&nbsp;Tuesday3 -&nbsp;Wednesday4 -&nbsp;Thursday5 - Friday6 - Saturday
You have to return the index for the day which is n&nbsp;days before the given day&nbsp;d.
Examples: 
Input: d = 4, n = 3
Output: 1
Explanation: 3 days before the 4th(Thursday) is 1(Monday).
Input: d = 2, n = 19
Output: 4
Explanation: 19 days before the 2nd(Tuesday) is 4(Thursday).
