# [Find one extra character](https://www.geeksforgeeks.org/problems/find-one-extra-character/1?page=5&category=Java&sortBy=submissions)
## Easy
Given two strings s1 and s2 which are of lengths n and n+1. The second string contains all the character of the first string, but there is one extra character. Your task to find the extra character in the second string and return it.&nbsp;
Examples:
Input: s1 = abcd, s2 = badce
Output: e
Input: s1 = efg, s2 = gtfe
Output: t
