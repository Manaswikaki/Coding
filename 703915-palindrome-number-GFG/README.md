# [Palindrome Number](https://www.geeksforgeeks.org/problems/palindrome0746/1)
## Easy
You are given an integer n. Your task is to determine whether it is a palindrome. A number is considered a palindrome if it reads the same backward as forward, as the number examples "12121" or "555".
Examples:
Input: n = 555
Output: trueExplanation: The number 555 reads the same backward as forward, so it is a palindrome.
Input: n = 123
Output: falseExplanation: The number 123 reads differently backward (321), so it is not a palindrome.
Input: n = -121
Output: trueExplanation: if number is palindrome, mainly ignore sign.
Constraints:-109&nbsp;≤ n ≤ 109
