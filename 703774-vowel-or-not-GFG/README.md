# [Vowel or Not](https://www.geeksforgeeks.org/problems/vowel-or-not0831/1?page=1&category=Java&sortBy=submissions)
## Easy
Given a English alphabet c, Write a program to check whether a character is a vowel or not.
&nbsp;
Example 1:
Input:
c = 'a'
Output:
YES
Explanation:
'a' is a vowel.
&nbsp;
Example 2:
Input:
c = 'Z'
Output:
NO
Explanation:
'Z' is not a vowel.
&nbsp;
&nbsp;
Your Task:
You don't need to read input or print anything. Your task is to complete the function isVowel() which takes a character c and returns 'YES' or 'NO'.
&nbsp;
Expected Time Complexity: O(1)Expected Auxiliary Space: O(1)
&nbsp;
Note: c is either lowercase or uppercase English alphabetic characterConstraints:
c belongs to English Alphabets.
&nbsp;
