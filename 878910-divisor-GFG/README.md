# [Divisor](https://www.geeksforgeeks.org/problems/divisor--125050/1?page=3&category=Java&sortBy=submissions)
## Easy
Given an integer n.&nbsp;Write a program to print all the divisors of n in a single line.
Examples:  
Input: n = 12
Output: 1 2 3 4 6 12
Explanation: 12 is divisible by 1 2 3 4 6 12.
Input: n = 10
Output: 1 2 5 10
Explanation: 10 is divisible by 1 2 5 10.
Constraints:1 &lt;= n &lt;= 105
